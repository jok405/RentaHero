# RentaHero
Superhero descriptions in Angular
## Dev Environment
  *Prerequisite: requires both node.js & npm *
### Installation
Install Angular CLI globablly: npm install -g @angular/cli
Install: npm install
### Launch Application
Within the project directory, launch the server by: ng serve --open
